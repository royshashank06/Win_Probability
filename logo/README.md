# IPL Win Predictor

🔗 **Live Demo:** [https://winprobability-rokomsdwall.streamlit.app](https://winprobability-rokomsdwall.streamlit.app)

A Streamlit app that predicts IPL match win probability based on live match situation (score, overs, wickets, target).

## Tech Stack
- Streamlit
- pandas, scikit-learn, xgboost
